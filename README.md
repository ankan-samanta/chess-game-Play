♟️ A full-stack real-time Chess Game developed using the MERN stack (MongoDB, Express.js, React.js, Node.js) and Socket.IO. The application enables two players to compete online with instant move updates, synchronized game states, and seamless communication through WebSockets. It includes chess rule validation, move history tracking, responsive design, and scalable backend architecture for real-time multiplayer gaming.

Features

Real-time multiplayer gameplay using Socket.IO
Chess move validation and legal move enforcement
Live game state synchronization
Responsive React UI
RESTful API with Express.js and Node.js
MongoDB database integration
Player matchmaking and room management
Move history and game tracking
Clean and modular MERN architecture

Tech Stack

Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB
Real-Time Communication: Socket.IO
State Management: React Hooks / Context API
Deployment: GitHub, Vercel/Render (optional)
